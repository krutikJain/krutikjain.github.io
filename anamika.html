What is Anamika?

Anamika is an all-in-one safety app for women, providing various types of features. It has a SOS trigger which can be triggered from app, home screen of device and by voice in which on single click fi rst call is dialed to police after hang-up automatically call is redirected to primary relative added by user and in background sends exact location via SMS to the contacts previously added by the user. Another feature “Helplines” provides the top emergency contacts like Police, Ambulance etc, from which the user can directly contact for help and can work offl ine as well. It off ers great UI and smooth User Experience.
