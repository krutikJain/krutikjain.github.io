What is Anamika?

Anamika is an all in one safety app for women, providing various types of features. It has a trigger alarm which sends exact location via SMS to the contact previously added by the user. It also triggers call to the first selected contact. Another feature “Helplines” provides the top emergency contacts like Police, Ambulance etc, from which the user can directly contact for help and Can work offline as well. It offers great UI and smooth User Experience.
Co Developed with Yaksh Bhesaniya.
